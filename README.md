# Java Basic Programs

This repository contains simple Java programs covering the basics of the language, perfect for beginners to understand syntax and core concepts.

## Programs Included

1. HelloWorld.java – Prints "Hello, World!" to the console.
2. VariablesExample.java – Demonstrates variables and data types.
3. OperatorsExample.java – Shows basic arithmetic operations.
4. IfElseExample.java – Checks if a number is even or odd.
5. LoopsExample.java – Demonstrates for, while, and do-while loops.
